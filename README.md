# ProgramacionWeb
Ejercicios de mi clase de Programacion Web

- Ejercicio 1: Se realizo un Hola Mundo en HTML5.
- Ejercicio 2: Se hizo uso de etiquetas y parrafos para aprender a dar presentación.
- Ejercicio 3: Se utilizaron enlaces y aprendimos el como funcionan.
- Ejercicio 4: Se inserto una imagen y se modifico.
- Ejercicio 5: Uso de etiquetas, enlaces, parrafos e imagenes.
- Ejercicio 6: Uso de <header>, <footer> y <section> para hacer secciones como pies de pagina y cabeceras.
- Ejercicio 7: Uso de <ul>, <ol> para listas ordenadas y desordenadas en este caso una receta.
- Ejercicio 8: Se realizo una tabla de Horarios de Clases con codigo CSS para el diseño.
- Ejercicio 9: Se creo un formulario usando validaciones y CSS.
- Ejercicio 10: Una Calculadora que convierte de °C a °F y usando JavaScript en donde se realizo las operaciones.
